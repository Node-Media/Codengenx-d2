# CodengenX - Web & App Solutions with Expert Training

A modern multi-page website for a web and app development company with comprehensive training programs, built with Next.js, TypeScript, and Tailwind CSS.

## Design Philosophy

- **No gradients**: Clean, solid colors with a distinctive palette
- **Unique color scheme**: Noir, Cream, Rust, Olive, and Slate for a sophisticated, modern look
- **Custom typography**: Space Grotesk for display and JetBrains Mono for accents
- **Bold geometric design**: Strong borders, layered elements, and brutalist-inspired aesthetics

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Pages

- **Home** - Hero, services overview, process, training highlights, testimonials
- **About Us** - Company story, mission/vision, team, core values, stats
- **Services** - Web development, mobile apps, product development, detailed training programs (long-term, short courses, 1-on-1 coaching)
- **Contact** - Contact form, office locations, R&D partnerships, FAQ
- **Blog** - Product development case studies, technical guides for web and mobile development

## Features

- Fully responsive design
- Modern, unique color palette (no typical gradients)
- Custom typography with Space Grotesk and JetBrains Mono
- Multi-page architecture with consistent navigation
- Comprehensive training program showcase
- R&D and innovation focus
- Product development case studies

## Tech Stack

- **Next.js 15** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Space Grotesk & JetBrains Mono** - Custom fonts

## Customization

Edit the color scheme in `tailwind.config.ts`:
```typescript
colors: {
  noir: "#0A0A0A",
  cream: "#F5F1E8",
  rust: "#C14B2A",
  olive: "#5C6B3E",
  slate: "#3D4852",
  sand: "#D4C5B0",
}
```
